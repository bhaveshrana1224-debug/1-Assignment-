#1-Assignment-

Primer designing for B2M homosapians. Go to NCBI gene, get the accession number for human B2M. Paste the accession number into the PCR template box in NCBI primer blast site. Set the parameters product size 80 to 200 BP, Temprature min 57℃, opt 60℃, max 62℃, GC% 40 to 60%, choose primer must span exon-exon junction if using cDNA.
